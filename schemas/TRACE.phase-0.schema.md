Real Phase 0 results:

```text

```

Placeholder result:

```text
not-run
```

Use `not-run` only when the scaffold or example has not recorded an actual prompt/tool run. A not-run TRACE placeholder is not execution evidence. Replace it with a real TRACE entry when the first run occurs.

Do not fabricate generated artifacts, tests, smoke-check results, accepted chunks, rejected chunks, or manual edits if they do not exist.
